---> O que é um Subprograma? 
Um subprograma é um bloco de código que realiza uma tarefa específica dentro de um programa maior. Ele pode ser chamado e executado sempre que for necessário, ao invés de fazer outro codigo voce so reutiliza 

Por exemplo, quando programamos em JavaScript (junto com HTML), ao criar um botão, normalmente associamos um evento a ele — como clicar para buscar dados. Para isso, dentro do JavaScript, criamos uma função que contém a operação para buscar esses dados.

Essa função é um subprograma: um pedaço de código separado que pode ser acionado (chamado) pelo evento do botão para executar uma tarefa específica, como buscar informações.

A function returns a value; in other words, it acts as a value that can be used in expressions or assigned to variables.

A procedure (or subroutine) performs a sequence of actions but does not return a value.
